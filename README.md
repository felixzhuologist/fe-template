# Frontend Template

Starter template with just the following:

* React
* Typescript
* HMR
* Tailwind CSS
  * cssnano for minifying in prod
* webpack, webpack-dev-server
* Prettier

Tested with the `node:14.15.1` image, e.g.
    
    docker run -it -v /yourdir:/code -p 8080:8080 node:14.15.1 /bin/bash
